README
=======

This repository contains work-in-progress **editors' drafts** of the Internet-Drafts being developed in the [IETF Tokbind Working Group](https://datatracker.ietf.org/wg/tokbind/charter/).

The following links yield HTML renderings of these **editors' drafts** (note also the spec name acronyms, please use them in Issue titles when submitting issues):
- TBNEGO: [draft-ietf-tokbind-negotiation-11](http://xml2rfc.ietf.org/cgi-bin/xml2rfc.cgi?modeAsFormat=html/ascii&url=https://raw.githubusercontent.com/TokenBinding/Internet-Drafts/master/draft-ietf-tokbind-negotiation-11.xml)
- TBPROTO: [draft-ietf-tokbind-protocol-17](http://xml2rfc.ietf.org/cgi-bin/xml2rfc.cgi?modeAsFormat=html/ascii&url=https://raw.githubusercontent.com/TokenBinding/Internet-Drafts/master/draft-ietf-tokbind-protocol-17.xml)
- HTTPSTB: [draft-ietf-tokbind-https-12](http://xml2rfc.ietf.org/cgi-bin/xml2rfc.cgi?modeAsFormat=html/ascii&url=https://raw.githubusercontent.com/TokenBinding/Internet-Drafts/master/draft-ietf-tokbind-https-12.xml)

Snapshots of the above, formally submitted to the [IETF Internet-Drafts repository](https://www.ietf.org/id-info/), are [here](https://datatracker.ietf.org/wg/tokbind/documents/).
